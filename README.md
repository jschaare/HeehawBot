# HeehawBot

## Running
1. Make .env file in directory with format (for now)
```
# .env
DISCORD_TOKEN={DISCORD SECRET TOKEN}
```
2. `pipenv install`
3. `pipenv shell`
4. `python -m bot`

## Notes
* To add packages `pipenv install <package>`
